## ADDED Requirements

### Requirement: Run Guide Document
The project SHALL provide a RUN_GUIDE.md document for developers.

#### Scenario: Developer starts backend
- **WHEN** developer follows backend startup steps
- **THEN** SpringBoot application runs on port 8080

#### Scenario: Developer starts frontend
- **WHEN** developer follows frontend startup steps
- **THEN** Vue 3 application runs on port 5173

#### Scenario: Developer verifies functionality
- **WHEN** developer follows verification steps
- **THEN** all MVP features are confirmed working
