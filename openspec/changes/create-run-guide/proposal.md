## Why

提供一份完整的前后端启动和验证文档，方便开发者快速运行和验证 MVP 系统。

## What Changes

创建 `RUN_GUIDE.md` 文档，包含：

1. **后端启动指南** - SpringBoot + H2
2. **前端启动指南** - Vue 3 + Vite
3. **功能验证步骤** - API 测试、页面验证
4. **常见问题处理**

## Capabilities

文档将涵盖：

| 模块 | 内容 |
|------|------|
| 后端 | Maven 启动、端口配置、H2 Console 访问 |
| 前端 | npm 启动、API 代理配置 |
| 验证 | curl 测试、页面功能验证 |
| 工具 | JaCoCo 覆盖率报告 |

## 技术栈

无新增技术栈，使用已有配置。
